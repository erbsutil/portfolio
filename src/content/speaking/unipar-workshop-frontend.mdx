Invited to lead a mini-course at UNIPAR's Academic Week for Analysis and Systems Development (TADS). The session simulated receiving a real task: start from scope and a prototype, decide the architecture, and start coding. We started with Astro components, then integrated React for the interactive parts — showing how JavaScript is only loaded where it's actually needed. One of the highlights was pulling data from the GitHub API directly in Astro's frontmatter, so the request happens at build time, not in the browser. We finished with a Netlify deploy and a quick look at SEO fundamentals and Page Speed Insights.
