Presented at DevParaná in Pato Branco. The talk moved beyond "what is Astro" to "why and how": comparing React bundle challenges against Astro's Islands approach, explaining how selective hydration works in practice, and showing a common mistake: the "Mega Island" trap where developers wrap an entire page in `client:load`, cancelling out all performance benefits. Included real build output comparisons between Next.js and Astro, and metrics showing the difference when islands architecture is applied correctly.
